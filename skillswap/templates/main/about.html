🔄 Про платформу SkillSwap

SkillSwap — це веб-система бартерного обміну навичками між людьми. Ніяких грошей. Тільки знання і час.

Концепція

Кожна людина щось вміє і щось хоче навчитися. SkillSwap з'єднує тих, чиї знання взаємодоповнюються: ти навчаєш когось Python — вони навчають тебе іспанської мови. Взаємовигідно.

Технічний стек

  • Backend: Python 3.12 / Flask (Application Factory)
  • ORM: Flask-SQLAlchemy + Flask-Migrate
  • Auth: Flask-Login + Werkzeug
  • Forms: Flask-WTF + CSRF protection
  • Frontend: Jinja2 SSR + Bootstrap 5.3 + Vanilla JS
  • DB: SQLite (за замовчуванням) / PostgreSQL
Приєднатися безкоштовно
