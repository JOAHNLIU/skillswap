# SkillSwap — можливості адміністратора

## Як отримати першого адміна

Перший зареєстрований користувач з `id=1` автоматично вважається суперадміном у логіці RBAC.

Також можна зробити користувача адміном через команду:

```bash
flask make-admin email@example.com
```

або через окремий файл:

```bash
python make_admin.py email@example.com
```

## Ролі

### Користувач
- реєстрація;
- профіль;
- навички;
- пошук інших користувачів;
- створення обмінів;
- чат;
- контракти;
- відгуки.

### Модератор
- перегляд скарг;
- обробка скарг;
- відповіді на тікети підтримки;
- верифікація користувачів.

### Адмін
- керування користувачами;
- керування навичками;
- керування обмінами;
- бан користувачів;
- верифікація профілів;
- перегляд аналітики;
- перегляд audit log.

### Суперадмін
- усе, що може адмін;
- призначення ролей іншим користувачам.

## Як надати роль через адмін-панель

1. Увійди під суперадміном.
2. Перейди в `Адмін → Користувачі`.
3. У рядку потрібного користувача знайди колонку `Надати роль`.
4. Обери роль у випадаючому списку:
   - Користувач;
   - Модератор;
   - Адмін;
   - Суперадмін.
5. Натисни `Надати`.

## Як забрати роль

У тому самому випадаючому списку вибери `Користувач` і натисни `Надати`.

## Інші можливості адміна

- ✅ Верифікувати користувача.
- 🚫 Заблокувати користувача.
- ✓ Розблокувати користувача.
- 🗑 Видалити користувача.
- Видаляти некоректні навички.
- Переглядати всі обміни.
- Працювати зі скаргами.
- Відповідати на звернення підтримки.
